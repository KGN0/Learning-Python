Introduction to Functions
- A function is a block of organized set of instructions that is used to perform a single task.
- Functions performs a single, specific, and well defined task.
- It provides better modularity and makes the code reusable
- A complicated block of monolithic code can be decomposed into more manageable pieces by using simpler functions and 
  this process is called functional decomposition

Functions can be categorized as follows
- Library functions or Built in Functions
- User defined functions
- Anonymous Functions
- Recursive Functions
- Modules

Need of Functions
- Clarity of code
- Code reuse 
- Reduce Errors

Function Defination: 
- A function is defined once, but invoked several times
- Function blocks begin with the keyword def followed by the function name and list of parameters(()) that it receives from callers
- Any input parameters or arguments should be placed within these parentheses.
- The function's body contains a set of indented statements which contains the code to execute when caller invoke the function

General Form of a Function Defination
- A function consists of two parts 
Syntax for function Defination:
def function_name(parameter1, parameter2, ....):
    documentation string
    statement block
    return
