1. Introduction to Python
    - Python is an open source, very simple, easy to read, general-purpose, dynamic, interpreted, powerful high programming language
    - Developed by a Dutch programmer named Guido van Rossum in 1991
    - Futher developed by Python Software Foundation
2. Why it is called 'Python'?
    - Python is named after the comedy television show Monty Python's Flying Circus and not named after the Python snake
3. The future of python?
    - Python has a huge user base that is constantly growing. It is a stable language that is going to stay for long.
    - The strength of python can be understood from the fact that this programming language is the most preferred language of companies.
